## A Anatomia de um Bom Prompt

Antes de entrar nos exemplos, vale entender o que torna um prompt realmente funcional. Não é magia — é estrutura.

**Papel** — diga ao modelo quem ele deve ser: "Você é um especialista em finanças pessoais" orienta o tom e o nível técnico da resposta.

**Contexto** — forneça o cenário: para quem é, qual o objetivo, quais restrições existem.

**Tarefa** — seja específico sobre o que você quer: redigir, resumir, listar, analisar, corrigir.

**Formato** — indique como a resposta deve ser apresentada: tópicos, tabela, parágrafo corrido, máximo de palavras.

Com esses quatro elementos, qualquer prompt fica mais preciso. Os modelos abaixo já incorporam essa lógica — você só precisa preencher as lacunas.

## Prompts para Trabalho e Produtividade

Estes prompts cobrem tarefas do dia a dia corporativo: e-mails, reuniões, relatórios e gestão de tempo.

```
Você é um assistente de produtividade. Escreva um e-mail profissional para [nome do destinatário] solicitando [objetivo do e-mail]. O tom deve ser [formal/cordial/direto]. Limite a 150 palavras.
```

```
Resuma a seguinte ata de reunião em até 5 pontos de ação, indicando responsável e prazo para cada um: [cole o texto da ata aqui]
```

```
Você é um gestor experiente. Crie uma pauta objetiva para uma reunião de [duração] sobre [tema]. Inclua tempo estimado para cada item.
```

```
Reescreva o seguinte parágrafo de relatório para soar mais profissional e claro, sem alterar o sentido: [cole o parágrafo]
```

```
Crie uma lista de tarefas priorizadas para a semana com base nos seguintes projetos em andamento: [liste os projetos e suas etapas]. Use o método de priorização por impacto e urgência.
```

```
Redija uma mensagem de feedback construtivo para um colega de equipe sobre [comportamento ou situação específica]. Seja direto, respeitoso e orientado a soluções.
```

```
Elabore um plano de 30/60/90 dias para um profissional recém-contratado como [cargo] em uma empresa de [setor].
```

## Prompts para Estudo e Aprendizado

![Pessoa estudando com laptop e anotações sobre IA](https://images.unsplash.com/photo-1498050108023-c5249f4df085?auto=format&fit=crop&w=1200&q=80)

Aprender com IA é mais eficaz quando você transforma o ChatGPT em um tutor personalizado. Se você está começando com IAs, o [guia completo sobre como usar o ChatGPT](/blog/como-usar-chatgpt-guia-completo-para-iniciantes) explica o básico antes de mergulhar nos prompts avançados.

```
Explique [conceito] como se eu tivesse 15 anos, sem jargões técnicos. Em seguida, dê um exemplo prático do dia a dia.
```

```
Crie um plano de estudos de [duração] para aprender [assunto ou habilidade] do zero. Inclua recursos gratuitos recomendados e uma meta semanal clara.
```

```
Gere 10 perguntas de múltipla escolha sobre [tema] para me ajudar a revisar antes de uma prova. Inclua a resposta correta e uma breve explicação de cada alternativa.
```

```
Você é um professor de [matéria]. Explique o conceito de [tópico específico] passo a passo, usando analogias simples e verificando minha compreensão ao final com uma pergunta.
```

```
Resuma o seguinte texto acadêmico nos principais pontos em até 300 palavras, mantendo os termos técnicos relevantes: [cole o texto]
```

```
Crie um mapa mental em formato de texto sobre [tema], com categorias principais e subcategorias, para facilitar minha memorização.
```

```
Elabore flashcards (pergunta/resposta) sobre [assunto] para revisão espaçada. Crie 10 flashcards com dificuldade progressiva.
```

## Prompts para Marketing e Redes Sociais

Marketing de conteúdo consome tempo — esses prompts aceleram a criação sem abrir mão da qualidade.

```
Você é um copywriter especializado em [nicho]. Escreva 5 opções de headline para um anúncio do produto [nome do produto], focando no benefício principal: [benefício]. Público-alvo: [descreva o público].
```

```
Crie um calendário editorial de 4 semanas para o Instagram de [tipo de negócio], com 3 posts por semana. Inclua tema, formato (carrossel/reels/estático) e chamada para ação de cada post.
```

```
Escreva uma legenda para Instagram sobre [tema do post] para uma marca de [segmento]. Tom: [descontraído/educativo/inspirador]. Inclua 10 hashtags relevantes ao final.
```

```
Você é um estrategista de conteúdo. Sugira 10 ideias de posts para o LinkedIn de um profissional de [área] que quer se posicionar como autoridade no tema [especialidade].
```

```
Crie um roteiro de 60 segundos para um Reels do Instagram apresentando [produto ou serviço]. Inclua gancho nos primeiros 3 segundos, problema, solução e CTA.
```

```
Escreva 3 variações de e-mail marketing para anunciar [lançamento/promoção/novidade] para uma lista de leads que já conhece a marca [nome da marca]. Destaque o benefício principal em cada versão.
```

## Prompts para Escrita e Conteúdo

![Teclado de laptop com código e texto em tela](https://images.unsplash.com/photo-1555066931-4365d14bab8c?auto=format&fit=crop&w=1200&q=80)

Seja para blog, roteiro ou ficção, estes prompts servem como ponto de partida sólido para qualquer projeto de escrita. Para automatizar a produção de conteúdo no seu fluxo, explore o [gerador de conteúdo com IA](/gerador).

```
Escreva uma introdução de blog (máx. 150 palavras) para um artigo sobre [tema]. A introdução deve apresentar o problema do leitor, prometer a solução e terminar com uma frase de transição para o corpo do texto.
```

```
Você é um editor experiente. Revise o texto a seguir quanto a clareza, coesão e gramática, explicando cada alteração: [cole o texto]
```

```
Crie um outline detalhado para um artigo de 1500 palavras sobre [tema], com H2, H3 e pontos principais de cada seção.
```

```
Escreva um parágrafo de conclusão para um artigo sobre [tema] que reforce a ideia central, traga uma reflexão final e incentive o leitor a [ação desejada].
```

```
Desenvolva um personagem para uma história de [gênero literário]. Inclua nome, idade, motivação principal, maior medo e um defeito que dificulta seus objetivos.
```

```
Reescreva o seguinte texto em um tom [mais informal/mais técnico/mais persuasivo], mantendo todas as informações originais: [cole o texto]
```

```
Crie 5 títulos alternativos para o artigo "[título atual]" com foco em CTR — use números, perguntas ou palavras de impacto.
```

## Prompts para Programação

Para devs, o ChatGPT funciona melhor como par — alguém que revisa, sugere e explica, não só escreve código por você.

```
Você é um desenvolvedor sênior em [linguagem de programação]. Revise o código abaixo, aponte problemas de performance, segurança ou legibilidade e sugira melhorias com explicação: [cole o código]
```

```
Explique o que o seguinte trecho de código faz, linha por linha, em linguagem simples para alguém que está aprendendo [linguagem]: [cole o código]
```

```
Escreva uma função em [linguagem] que [descrição da funcionalidade]. Inclua comentários explicando cada etapa e um exemplo de uso no final.
```

```
Crie testes unitários para a seguinte função em [linguagem/framework de testes]. Cubra os casos: entrada válida, entrada inválida e edge cases: [cole a função]
```

```
Você é um especialista em [banco de dados]. Otimize a seguinte query SQL para melhorar a performance em uma tabela com [número] de registros, explicando cada mudança: [cole a query]
```

```
Converta o seguinte código de [linguagem A] para [linguagem B], mantendo a mesma lógica e adicionando comentários onde a sintaxe for muito diferente: [cole o código]
```

```
Liste os 5 erros mais comuns que iniciantes cometem ao trabalhar com [tecnologia/framework] e como evitar cada um deles.
```

## Prompts para Negócios e Empreendedorismo

Ao avaliar qual IA usar para esses cenários de negócio, vale conferir a comparação completa entre [ChatGPT vs Gemini vs Claude](/blog/chatgpt-vs-gemini-vs-claude-qual-a-melhor-ia-em-2026) — cada modelo tem pontos fortes distintos.

```

```

```
Elabore uma proposta comercial profissional para oferecer [serviço ou produto] a [tipo de cliente]. Inclua apresentação da empresa, benefícios, entregáveis, prazo e forma de pagamento.
```


## Dicas para Adaptar os Prompts

**Seja específico nos colchetes.** Quanto mais contexto você coloca nas variáveis, melhor a resposta. "Público-alvo: [mães de 30-45 anos que trabalham fora]" rende muito mais do que "[público]".

**Itere sobre a resposta.** Não descarte uma resposta ruim — peça ao ChatGPT para ajustar: "Refaça o item 3 com um tom mais direto" ou "Adicione exemplos práticos à segunda seção".

**Combine prompts.** Use o resultado de um prompt como entrada para outro. Gere o outline com um prompt e o texto completo com outro, seção por seção.

**Salve seus melhores prompts.** Crie um documento pessoal com os modelos que funcionaram bem para o seu contexto. Com o tempo, você vai construir uma biblioteca particular muito mais precisa do que qualquer lista genérica. Aliás, você também pode explorar as opções gratuitas com as [melhores IAs gratuitas](/blog/melhores-ias-gratuitas-2026) para testar esses prompts em diferentes modelos.

## Perguntas Frequentes

### Os prompts funcionam em outros modelos além do ChatGPT?

Sim. A maioria dos prompts desta lista funciona bem no Gemini, Claude e outros modelos de linguagem modernos. Pequenos ajustes de tom podem ser necessários — o Claude, por exemplo, tende a seguir instruções de formato com mais precisão, enquanto o Gemini responde bem a contextos com dados factuais.

### Qual é o tamanho ideal de um prompt?

Não existe um número mágico, mas prompts entre 50 e 200 palavras tendem a produzir os melhores resultados para tarefas complexas. Para tarefas simples (como reescrever uma frase), prompts curtos funcionam perfeitamente. O que importa é que todos os elementos essenciais estejam presentes: papel, contexto, tarefa e formato.

### Como evitar respostas genéricas e superficiais?

O principal motivo de respostas rasas é a falta de contexto. Adicione detalhes: setor da empresa, tom de voz da marca, nível de conhecimento do público, restrições de tamanho. Outra estratégia eficaz é pedir ao modelo que faça perguntas antes de responder — isso força um alinhamento antes da execução.

### Posso usar esses prompts para fins comerciais?

Sim. As respostas geradas pelo ChatGPT podem ser usadas comercialmente de acordo com os termos de uso da OpenAI. O conteúdo gerado é de responsabilidade do usuário, então sempre revise, adapte e adicione sua perspectiva antes de publicar ou entregar para clientes.
